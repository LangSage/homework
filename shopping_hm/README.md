# Shopping Homework

Static homework app for the shopping topic.

- Main page: `index.html`
- Assignment code sent to Google Forms: `HOMEFORK_TG25`
- Grammar tasks: 3 of 24
- Progress storage: `localStorage` plus a small cookie summary
- Offline assets: service worker cache
- Dictionary: `words.json`, with canvas word hotspots
- Audio: neural Edge TTS MP3 clips

Regenerate images and audio on Windows:

```powershell
python .\tools\generate_assets.py
```
